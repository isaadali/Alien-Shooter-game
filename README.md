# 👾 Alien Shooter Game (C++)

## 📌 Overview

Alien Shooter is a simple yet engaging shooting game developed in C++. The player controls a character that shoots incoming alien enemies while trying to survive and achieve the highest score.

 Features

* Player movement and shooting controls
* Multiple alien enemies
* Collision detection system
* Score tracking
* Game over condition



 Technologies Used

* C++


 How to Run

1. Clone the repository
2. Open the project in your IDE (e.g., VS Code)
3. Compile the code using a C++ compiler
4. Run the executable file



 Game Objective

* Shoot aliens to earn points
* Avoid enemy attacks
* Survive as long as possible
